2. Что такое регистры в 1С

Регистр — это внутреннее место хранения учётных данных в системе 1С. Именно туда документ записывает результат после проведения.

Проще говоря, документ оформляет хозяйственную операцию, а регистр запоминает её результат: сколько товара пришло, сколько продано, сколько денег поступило или выдано.

Если документ не проведён, движения по регистрам не формируются, а значит остатки и отчёты не меняются.

3. Главная логика темы

Документ → Проведение → Движения по регистрам → Остатки → Отчёт

После проведения документ делает движения по регистрам. Эти движения изменяют учётные показатели, а на их основе строятся остатки, обороты и отчёты.

4. Что делают регистры после проведения

Документ Какие движения возникают Что меняется Где видно результат
Приход товара Запись в регистр остатков товаров Количество товара увеличивается Остатки на складе, отчёт по товарам
Реализация товара Списание из регистра остатков Количество товара уменьшается Продажи, остатки, обороты
ПКО Запись в регистр кассы Сумма денег в кассе увеличивается Кассовый отчёт
РКО Списание из регистра кассы Сумма денег в кассе уменьшается Кассовый отчёт

5. Как проходит движение данных

1. Выбрать данные
2. Создать документ
3. Заполнить реквизиты
4. Провести
5. Создать движения
6. Изменить остатки
7. Получить отчёт

Что важно понять студенту

  • Документ сам по себе — это форма операции.
  • Проведение запускает запись данных в регистры.
  • Регистры хранят движение и остатки.
  • Отчёты строятся не из формы документа, а из данных регистров.
