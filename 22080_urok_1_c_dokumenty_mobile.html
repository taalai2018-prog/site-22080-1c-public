2. Что такое документ 1С

Документ — это запись конкретной хозяйственной операции в системе 1С.

Например: приход товара, реализация, кассовый ордер, перемещение, списание.

Документ показывает не постоянные данные, а то, что произошло в организации.

3. Главная логика темы

Справочник → Документ → Проведение → Отчёт

Документ использует данные из справочников, после проведения влияет на учёт, а результат виден в отчёте.

4. Чем документ отличается от справочника

Понятие Что хранит Пример Назначение
Справочник Постоянные данные Товар, поставщик, склад Основа для заполнения документов
Документ Конкретную операцию Приход, продажа, ПКО Отражает событие в учёте

5. Основные виды документов 1С

Документ Что отражает Какие справочники использует Результат
Приход товара Поступление товара Номенклатура, Контрагенты, Склады Увеличение остатка на складе
Реализация Продажа товара Номенклатура, Контрагенты, Склады Уменьшение остатка, отражение продажи
ПКО Поступление наличных денег Кассы, Контрагенты Увеличение суммы в кассе
РКО Выдача наличных денег Кассы, Контрагенты Уменьшение суммы в кассе
Перемещение Перенос товара между складами Номенклатура, Склады Изменение остатков по складам

Что такое реквизиты документа

Реквизиты документа — это поля, которые нужно заполнить.

  • Дата
  • Номер
  • Товар
  • Поставщик или покупатель
  • Склад
  • Количество
  • Цена
  • Сумма
