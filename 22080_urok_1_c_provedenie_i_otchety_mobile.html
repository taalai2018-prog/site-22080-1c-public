2. Что такое проведение документа

Проведение — это момент, когда документ не просто сохраняется, а начинает влиять на учёт в системе 1С.

До проведения документ может быть только черновиком. После проведения он изменяет остатки, суммы, кассу, склад или другие учётные данные.

Например, документ «Приход товара» после проведения увеличивает остаток товара на складе.

3. Главная логика темы

Справочник → Документ → Проведение → Отчёт

Сначала выбираются данные из справочников, потом создаётся документ, после проведения он отражается в учёте, а затем виден в отчёте.

5. Как документ попадает в отчёт

1. Выбрать данные
2. Создать документ
3. Заполнить реквизиты
4. Сохранить
5. Провести
6. Открыть отчёт
7. Проверить результат

Что важно понять студенту

  • Без проведения документ не влияет на учёт.
  • Проведённый документ меняет данные в системе.
  • Отчёт показывает итог уже проведённых документов.
  • Ошибочно проведённый документ даёт неправильный отчёт.
